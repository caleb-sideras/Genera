Credits

Users will be given 50 free credits on account creation

Credits allow users to perform actions on Genera. Each action performed will use 1 credit, which include:

  • NFT Generation (w/ Metadata)
  • NFT IFPS Storage/Minting

Example w/ 50 credits

  • Generate 50 NFTs or
  • Generate 25 NFTs Mint 25 NFTs (w/ IFPS deployment)
  • Mint 50 NFTs (w/ IFPS deployment)
